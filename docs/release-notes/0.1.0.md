# Fishing Net Mod 0.1.0 Release Notes

## Status

0.1.0 is a test release focused on the core single-player gameplay loop.

## Included Features

- Four fishing net tiers: copper, silver, gold, and iridium.
- SMAPI debug command: `fishing_net give <copper|iron|gold|iridium>`.
- Active net casting into water with stamina cost and a number challenge.
- Passive net placement in water with daily production and harvest challenge.
- Content Patcher items, crafting recipes, mail, and custom net sprites.
- Quest-style recipe progression for copper, second-tier, gold, and iridium nets.

## Installation

1. Install SMAPI 4.x.
2. Install Content Patcher 2.x or newer.
3. Extract `FishingNetMod 0.1.0.zip` into the Stardew Valley `Mods` folder.
4. Launch the game through SMAPI.

Expected extracted layout:

```text
Mods/FishingNetMod/FishingNetMod/manifest.json
Mods/FishingNetMod/FishingNetMod/FishingNetMod.dll
Mods/FishingNetMod/[CP] FishingNetMod/manifest.json
Mods/FishingNetMod/[CP] FishingNetMod/content.json
```

## Verification

- Manual test record: `docs/reports/2026-06-08-0.1.0-manual-test-results.md`.
- Build artifact: `FishingNetMod/bin/Debug/net6.0/FishingNetMod 0.1.0.zip`.

## Known Limits

- Multiplayer support is experimental. 0.1.0 is validated primarily for single-player.
- Text is not fully localized. Content Patcher item/mail text is English, while several C# HUD/error messages are Chinese.
- Active casting has basic feedback only: water detection, stamina cost, sound, challenge menu, and HUD result.
- Passive net production uses the game's fish generation behavior and does not enforce a separate fixed 10% trash chance.
- Harvesting uses the number challenge and direct item return, not a chest-style harvest UI.
